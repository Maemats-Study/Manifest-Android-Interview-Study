# Material Design Components(MDC)란?

- Google의 Material Design 가이드라인을 기반으로 하는 커스텀 가능한 UI 위젯 및 컴포넌트 집합
- 해당 컴포넌트는 개발자가 앱의 브랜딩 및 디자인 요구사항에 맞게 모양과 동작을 커스텀할 수 있도록 하면서 일관되고 사용자 친화적인 인터페이스를 제공하도록 설계됨

## Material Design Components의 주요 특징

1. Meterial Theming
    - 테마 설정 지원 → 개발자가 타이포그래피, 모양 및 색상을 전역적으로 도는 컴포넌트 수준에서 커스텀할 수 있도록 함
2. 미리 빌드된 UI 컴포넌트
    - 버튼, 카드, 앱 바, 내비게이션 드로어, 칩 등과 같이 즉시 사용할 수 있는 광범위한 UI 컴포넌트를 제공
    - 접근성, 성능 및 반응성에 최적화
3. 애니메이션 지원
    - Material Design은 모션 및 전환 강조
    - MDC에는 공유 요소 전환, 리플 효과 및 시각적 피드백과 같은 애니메이션에 대한 내장 지원이 포함되어 있어 사용자 상호 작용을 향상시킴용을
4. 다크 모드 지원
    - 다크 모드를 쉽게 구현할 수 있는 API가 포함되어 있어 시각적 일관성을 보장하면서 라이트 및 다크 모드에 대한 테마를 정의할 수 있음
5. 접근성
    - MDC는 더 큰 터치 대상, 접근성을 위한 sementic 레이블 및 적절한 포커스 관리와 같은 기능을 제공 → 접근성 표준 준수
    - 모든 사용자를 위한 포괄적인 UI를 보장
