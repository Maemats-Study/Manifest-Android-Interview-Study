# Q) 50. Material Design Components (MDC)란 무엇인가요?
커스텀 가능한 UI 위젯 및 컴포넌트 집합입니다.
개발자가 앱의 브랜딩 및 디자인 요구 사항에 맞게 모양과 동작을 커스텀할 수 있도록 하면서 일관되고
사용자 친화적인 인터페이스를 제공하도록 설계

[MDC Library 링크](https://github.com/material-components/material-components-android)

### 주요 특징
1. Material Theming<br/>
2. 미리 빌드된 UI 컴포넌트
3. 애니메이션 지원
4. 다크 모드 지원
5. 접근성<br/>
접근성 향상을 위한 sementic 레이블 및 적절한 포커스 관리와 같은 기능을 제공
접근성 표준을 준수함.

### 실전 질문<br/>
Q) MDC의 Material Theming은 앱 전체에서 디자인 일관성을 유지하는 데 어떻게 도움이 되나요?<br/>
테마 설정을 지원하여 개발자가 타이포그래피, 모양 및 색상을 전역적으로 또는
컴포넌트 수준에서 커스텀할 수 있습니다.
