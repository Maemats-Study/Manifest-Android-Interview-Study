# Q) 54. Jetpack ViewModel에 대해 설명해 주세요.
Jetpack ViewModel 은 생명주기를 인식하는 방식으로 UI 관련 데이터를 저장하고
관리하도록 설계된 안드로이드 아키텍처 컴포넌트의 핵심 구성 요소

안드로이드 컴포넌트의 생명주기를 인식하고 MVVM 아키텍처 패턴과 효과적으로 통합되어
화면 회전과 같은 이벤트 중에 데이터를 유지함으로써 상태 관리를 단순화하고 전반적인 개발 경험을 향상

### ViewModel 특징
Jetpack activity-ktx 라이브러리 에서 제공하는 ComponentActivity 의 확장함수인
`viewModels()` 델리게이트를 사용해 ViewModel 을 쉽게 생성할 수 있습니다.

```kotlin
private val viewModel: MyViewModel by viewModels()
```

### Jetpack ViewModel 과 MVVM ViewModle 차이
Jetpack ViewMoel:<br/>
UI 관련 상태를 유지하는 데 의의를 둔 방식으로, UI 상태관리에 초점을 두었습니다.

MVVM ViewModel:<br/>
View 와 Model 간의 중개자 역할을 하며 바인딩을 처리하고 UI 가 ViewModel 에서 제공하는 데이터를
표시하는 데만 관련되도록 보장합니다.
조금더, 아키텍처 패턴에 대한 중점.
