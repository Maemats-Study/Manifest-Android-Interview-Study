# Q) 49. AppCompat 라이브러리란 무엇인가요?

AppCompat 라이브러리:<br/>
안드로이드 하위 버전과의 호환성을 유지하는 데 도움이 되도록 설계된 Android Jetpack 제품군의 일부

### 주요 기능
1. UI 컴포넌트 하위 호환성 제공<br/>
2. Material Design 지원<br/>
3. 테마 및 스타일링 지원<br/>
4. 동적 기능지원<br/>
동적 리소스 로딩 및 벡터 드로어블 지원을 제공

### 사용이유
최신 안드로이드 기능과 UI 컴포넌트가 지원되는 모든 API 레벨에서 일관되게 작동되도록 보장하기 위해서 사용

### 실전 질문
Q) AppCompat 라이브러리는 하위 안드로이드 버전에서 Material Design 지원을
어떻게 가능하게 하며, 이와 같은 동작을 기반으로 하는 주요 UI 컴포넌트에는 무엇이 있나요?<br/>
AppCompat는 테마 + 위젯 대체 + 틴팅/벡터 호환으로 구형 버전에서도 머터리얼 룩&필을 구현합니다.
