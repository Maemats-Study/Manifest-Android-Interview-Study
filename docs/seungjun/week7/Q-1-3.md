# Q) 3. recomposition이란 무엇이며 언제 발생하나요? 또한 앱 성능과 어떤 관련이 있나요?

### 리컴포지션 발생 조건

1. **매개변수에 변경이 발생했을 때(Input Changes):**

   컴포저블 함수는 입력 매개변수가 변경될 때 recompsition 을 트리거한다.

   `quals()` 함수를 사용해 새 매개변수 값을 이전 값과 비교, 비교 결과가 false 이면 런타임에서

   변경사항을 인지해 recompsition 을 트리거하여 동기화가 필요한 부분에 한해서 UI 업데이트

2. **상태 변경이 관찰되었을때(Observing State Changes):**

   `remember` 함수와 `State API` 를 함께 사용해 상태 변경을 모니터링

   상태 객체를 메모리에 보존하여, recomposition 시 메모리에 저장된 값을 복원해 UI 최신 상태를 일관되게 반영하도록 보장
